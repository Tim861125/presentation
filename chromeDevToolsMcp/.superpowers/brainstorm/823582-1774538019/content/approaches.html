三個改版方向

選一個最符合你想要的技術分享風格

A

Prompt Cookbook

以「怎麼寫 prompt」為主軸。每類工具附上真實 prompt 範例，從基礎到進階，讓 RD 帶走一份可直接複用的 prompt 模板集。

結構：簡介 → Prompt 設計原則 → 四大類工具 prompt 範例 → 串接範例 → 總結

B

Case Study 驅動

以 2 個真實場景為骨幹（例：LCP 慢、API 異常排查），完整走一遍 AI 輔助的調查流程，prompt 跟工具在故事脈絡中自然出現。

結構：簡介 → 場景 1 完整流程 → 場景 2 完整流程 → 抽取通用模式 → 總結

C

混合：Prompt 原則 + 場景演示

前半教 prompt 寫法（原則 + 範例），後半用一個完整 case study 串起所有工具，兼顧「學到方法」和「看到實戰」。

結構：簡介 → Prompt 原則（附範例）→ 工具組合技 → 完整案例走一遍 → 總結
