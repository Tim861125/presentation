# MCP 技術分享報告規格

## 此專案使用 slidev 呈現報告
- 注意版面長度，不要超過螢幕長度

## 首頁格式

```
theme: default
background: https://images.unsplash.com/photo-1639322537228-f710d846310a?q=80&w=2832
class: text-center
highlighter: shiki
lineNumbers: false
```

## 內容規範
- 簡潔有力
- 不要使用過多 icon 呈現
- 使用簡潔的範例呈現
- 報告內容大約 20 頁
- 內容不用使用 click 功能
